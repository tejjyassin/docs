# Page vendre

The "Page vendre" is a dedicated promotional landing page aimed at providing a service to promote properties (bien) for our customers. Within this page, a range of distinct components awaits exploration, as we will discuss in this section. {{ className: 'lead' }}


## Titre de la page

<Row>
  <Col>

    Titre de la page is a simple component that displays the title and subtitle of the page. It is used on the "Vendre" page to display the title of the page.

    ### Available settings

<Properties>
  <Property name="sous titre" type="text">
    The subtitle of the section
  </Property>
  <Property name="titre" type="text">
    The title of the section
  </Property>

## Bloc de texte avec quatre images

<Row>
  <Col>

   Bloc de texte avec quatre images is an advanced component that displays a text block with four images. It is used on the "Vendre" page to display the title, description of our network and how we can be helpful as a company to promote the customers' listing

    ### Available settings

<Properties>
  <Property name="Titre" type="text">
    The title of the section
  </Property>
  <Property name="Prémiere paragraph" type="texte">
    The first paragraph of the section
  </Property>

  <Property name="Deuxième paragraph" type="texte">
    The second paragraph of the section
  </Property>

  <Property name="Galerie des images" type="images">
    The four images for the section
  </Property>
  
  
</Properties>

  </Col>
  <Col>

## Fonctionalites

<Row>
  <Col>

   A dedicated block to display the features of our service in a three column design. 

    ### Available settings

<Properties>
  <Property name="Sous titre" type="texte">
    The subtitle of the section
  </Property>
  <Property name="Titre" type="texte">
    The title of the section
  </Property>  

  <Property name="Description" type="texte">
    The description of the section
  </Property>  

  <Property name="Fonctions" type="texte">
    The features to be display, it contain the following details : 
    1. **Nom** : The title of the feature
    2. **Description** : The description of the feature
    3. **Lien** : The link to the feature
  </Property>  
  
</Properties>

  </Col>
