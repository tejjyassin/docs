# Page Vendre

La page "Page Vendre" est une page de destination promotionnelle dédiée visant à fournir un service pour promouvoir les biens de nos clients. À l'intérieur de cette page, une gamme de composants distincts attend d'être explorée, comme nous le discuterons dans cette section. {{ className: 'lead' }}


## Titre de la page

<Row>
  <Col>

    Le "Titre de la page" est un composant simple qui affiche le titre et le sous-titre de la page. Il est utilisé sur la page "Vendre" pour afficher le titre de la page.

    ### Paramètres disponibles

<Properties>
  <Property name="sous titre" type="texte">
    Le sous-titre de la section
  </Property>
  <Property name="titre" type="texte">
    Le titre de la section
  </Property>
  
  
</Properties>

## Fonctionnalités

<Row>
  <Col>

   Un bloc dédié pour afficher les fonctionnalités de notre service dans une conception à trois colonnes. 

    ### Paramètres disponibles

<Properties>
  <Property name="Sous titre" type="texte">
    Le sous-titre de la section
  </Property>
  <Property name="Titre" type="texte">
    Le titre de la section
  </Property>  

  <Property name="Description" type="texte">
    La description de la section
  </Property>  

  <Property name="Fonctions" type="texte">
    Les fonctionnalités à afficher, elles contiennent les détails suivants : 
    1. **Nom** : Le titre de la fonctionnalité
    2. **Description** : La description de la fonctionnalité
    3. **Lien** : Le lien vers la fonctionnalité
  </Property>  
  
</Properties>

  </Col>
