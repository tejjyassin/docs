## Bloc de fonctionalites

<Row>
  <Col>

    Bloc de fonctionalites is a component that displays some data in a three column design. numbers and values on the left and the right and an image at the center.

    ### Available settings

<Properties>
  <Property name="Image" type="image">
    The centered image of the section
  </Property>
  <Property name="fonctionalites" type="text">
    This section include the following details : 
    1. **alignment** : Where do you want to display the data, left or right
    2. **numéro** : The number of the data
    3. **titre** : The value of the data
    4. **icon** : The icon of the data
  </Property> 
  
</Properties>

  </Col>

## Promotion en deux bloc

<Row>
  <Col>

   The "Promotion en deux bloc" is a component that presents a three-column design. The initial column showcases the first promotion, the second column features an image, and the third column highlights the second promotion. In our scenario, we utilize this component to promote two services: one for clients interested in purchasing a property and another for clients interested in selling a property.

    ### Available settings

<Properties>
  <Property name="Gauche" type="groupe">
    Contain a group of fields : 
    1. **Titre** : The title of the column
    2. **Description** : The description of the column
    3. **Lien** : The link of the column
    4. **Texte du lien** : The text of the link
  </Property>
  <Property name="Image" type="image">
    The centered image of the section
  </Property> 

  <Property name="Droits" type="groupe">
    Contain a group of fields : 
    1. **Titre** : The title of the column
    2. **Description** : The description of the column
    3. **Lien** : The link of the column
    4. **Texte du lien** : The text of the link
  </Property>
  
</Properties>

  </Col>
  <Col>

## Equipe

<Row>
  <Col>

    The "Equipe" is a component that presents a three-column design. Each column features a team member's image, name, and title. In our scenario, we utilize this component to showcase the team members of the company.

    ### Available settings

<Properties>
  <Property name="Titre" type="texte">
    The title of the section
  </Property>
  <Property name="Equipe" type="groupe">
    Contain a group of fields : 
    1. **Image** : The image of the team member
    2. **Nom** : The name of the team member
    3. **Position** : The position of the team member
  </Property> 
  
</Properties>
