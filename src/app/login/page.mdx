# Connexion & Tableau de bord

Dans cette section de la documentation, nous vous guiderons à travers les étapes essentielles pour vous connecter à votre site WordPress. La connexion est la première étape pour accéder à l'interface d'administration de votre site, où vous pouvez gérer le contenu, personnaliser l'apparence de votre site et effectuer diverses tâches administratives. {{ className: 'lead' }}

## Connexion

La première étape pour vous connecter se fait via l'URL WordPress par défaut, et les informations d'identification nécessaires vous seront envoyées par e-mail. Ces informations d'identification de connexion uniques ont été créées pour chaque utilisateur afin de garantir un accès sécurisé à leur espace administratif respectif. Si vous rencontrez des difficultés lors de l'utilisation de ces informations d'identification ou si vous avez besoin d'une assistance supplémentaire, n'hésitez pas à <a href="https://synergie-media.com/contact/" target="_blank">nous contacter</a>. Nous sommes là pour vous aider avec tout problème lié à la connexion.

![Connexion](/screenshots/login.png)

<Note>
Veuillez ne pas partager votre mot de passe avec qui que ce soit d'autre ! Si vous avez besoin de donner à quelqu'un l'accès à votre site, vous pouvez créer un nouvel utilisateur pour lui avec un ensemble différent de permissions.

</Note>

## Tableau de bord

Après vous être connecté, vous serez accueilli par votre tableau de bord, qui sert de centre de commande pour votre site web, vous permettant de gérer et de mettre à jour son contenu.

![Tableau de bord](/screenshots/dashboard.png)

Le tableau de bord est organisé en plusieurs sections :

1. **Articles** : Cette section est dédiée à la gestion de tous les articles liés à la section actualités de votre site web. Ici, vous pouvez créer, modifier ou supprimer des articles qui apparaissent dans la section articles de votre site.

2. **Médias** : Cette zone vous permet de gérer vos fichiers multimédias.

3. **Pages** : Dans cette section, vous avez le contrôle sur les différentes pages affichées sur votre site web. Les pages englobent tous les liens visibles sur votre site, à l'exception des articles et des annonces immobilières, qui ont leurs propres sections dédiées à la gestion.

4. **Commentaires** : Veuillez ignorer cette section, car nous n'utilisons pas de commentaires sur notre site web.

5. **Biens** : C'est ici que vous pouvez gérer toutes les annonces immobilières visibles sur le site web. Vous pouvez également contrôler leurs types et leurs classifications.

6. **Profil** : Dans cette section, vous pouvez mettre à jour vos informations d'identification de connexion, y compris votre nom d'utilisateur et votre mot de passe.

7. **Outils** : Bien que WordPress fournisse cette section par défaut, elle n'est pas applicable dans notre cas. Comme mentionné précédemment, nous utilisons une technologie différente pour la partie visible du site, et l'interface d'administration a été simplifiée pour faciliter uniquement les mises à jour de contenu nécessaires.

8. **Global** : Cette section revêt une grande importance, car elle vous permet d'apporter des modifications à tous les éléments récurrents du site web. Ici, vous pouvez mettre à jour les liens généraux et les phrases répétées sur tout le site pour maintenir la cohérence et simplifier la gestion du contenu.
