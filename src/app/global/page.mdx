# Paramètres globaux

Les paramètres globaux, comme leur nom l'indique, regroupent une section dédiée comprenant toutes les configurations générales ou récurrentes du site web. Dans cette section, nous explorerons certaines de ces configurations essentielles. {{className:'lead'  }}

## Structure des paramètres globaux

![Paramètres globaux](/screenshots/global.png)*Globale*

Les paramètres globaux sont divisés en quatre onglets principaux, chacun contenant un ensemble de configurations. Ces onglets sont :

## 01. En-tête (Header)

<Row>
<Col>
Contient des détails sur la section de l'en-tête du site web et comment elle doit être affichée. La section de l'en-tête est la section la plus haute du site web, qui contient le logo, le menu de navigation et d'autres détails essentiels. La section de l'en-tête est généralement la première chose qu'un visiteur voit lorsqu'il visite un site web. Il est donc essentiel de la configurer correctement.
</Col>

<Col>

![En-tête](/screenshots/header.png)*En-tête*

</Col>

</Row>

<Row>
<Col>

La section du pied de page est la section la plus basse du site web, qui contient le logo, le menu de navigation et d'autres détails essentiels. La section du pied de page est généralement la dernière chose qu'un visiteur voit lorsqu'il visite un site web. Il est donc essentiel de la configurer correctement.

## 03. Contenu (Contenu)

<Row>
<Col>

Le contenu est l'endroit où vous trouverez tout le contenu général du site web, des phrases répétées telles que "Plus d'informations", "Lire la suite", etc. Gardez à l'esprit que ces phrases sont répétées sur l'ensemble du site web, donc toute modification sera reflétée partout.

</Col>
