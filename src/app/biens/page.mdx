# Biens

La section "Biens" sert de zone spécialisée pour toutes les annonces disponibles sur le site web. Tout comme les <a href="/pages">pages</a> et les <a href="/posts">articles</a>, vous pouvez effectuer des actions telles que l'ajout, la modification et la suppression d'un "bien" de manière similaire. Cependant, ce qui distingue les "Biens" est leur structure de contenu unique. Alors que les articles utilisent un éditeur simple pour la saisie du contenu et que les pages reposent sur le système de <a href="/components">composants</a>, les "Biens" utilisent un système distinct, que nous examinerons dans cette section.

<Note>
Ajouter/modifier et supprimer un nouveau bien est similaire à la structure des pages/articles, si vous avez des difficultés avec les bases, veuillez vous référer aux sections <a href="/pages">pages</a> et <a href="/posts">articles</a>.
</Note>

## Structure d'un Bien

Pour ajouter un nouveau bien, vous devez vous rendre dans la section "Biens" du menu de gauche. Ensuite, cliquez sur "Ajouter un bien" en haut à gauche.

Vous obtiendrez une nouvelle page avec un titre, une liste de formulaires et une barre latérale contenant des informations supplémentaires.

![Ajouter un nouvel article](/screenshots/biens.png)*Biens -> Ajouter un nouveau Bien*

La liste des formulaires est divisée en onglets, chaque onglet contient une liste de champs que vous pouvez remplir. Voici une liste des onglets disponibles :

## 01. Générale

<Row>
<Col>
L'onglet "Générale" contient des informations générales sur le bien. Voici une liste détaillée de tous ses champs :

1. **Vente** : Un interrupteur indiquant si le bien est à vendre ou non.
2. **Location** : Un interrupteur indiquant si le bien est à louer ou non.
3. **Location meublée** : Un interrupteur indiquant si le bien est à louer meublé ou non.
4. **Location vide** : Un interrupteur indiquant si le bien est à louer vide ou non.
5. **Location saisonnière** : Un interrupteur indiquant si le bien est à louer saisonnier ou non.
6. **Exclusif** : Un interrupteur indiquant si le bien est exclusif ou non.
7. **Projets Neufs** : Un interrupteur indiquant si le bien est un nouveau projet ou non.
8. **Statut** : Une liste d'options pour indiquer le statut du bien, les options disponibles sont : Publiée (par défaut), Louée, Vendue.
9. **Référence** : Un champ texte pour indiquer la référence du bien.
10. **id_bien** : Un champ texte pour indiquer l'identifiant du bien (utilisé comme référence sur l'ancien site web).
11. **Titre en Français** : Un champ texte pour indiquer le titre du bien en français.
12. **Titre en Anglais** : Un champ texte pour indiquer le titre du bien en anglais.
13. **Description en Français** : Un champ texte pour indiquer la description du bien en français.
14. **Description en Anglais** : Un champ texte pour indiquer la description du bien en anglais.
</Col>
<Col>

![Paramètres généraux](/screenshots/general.png)*Biens -> Ajouter un nouveau Bien -> Générale*
![Général sur le site web](/screenshots/biens-general.png)*Exemple d'informations générales sur un bien*
</Col>
</Row>

## 03. Prix

<Row>
<Col>
L'onglet "Prix" contient le prix du bien. Voici une liste détaillée de tous ses champs :

1. **Prix de vente** : Un champ texte pour indiquer le prix de vente du bien.
2. **Prix de location par mois** : Un champ texte pour indiquer le prix de location par mois.
3. **Prix de location par jour** : Un champ texte pour indiquer le prix de location par jour.
4. **Nous consulter** : Un interrupteur indiquant si le prix est sur demande ou non.
5. **Frais de syndic** : Un champ texte pour indiquer les frais de syndic.
6. **Prix ancien** : Un champ texte pour indiquer l'ancien prix du bien (non utilisé sur le nouveau site web).
</Col>

L'onglet "Coordonnées" contient les coordonnées du bien. Voici une liste détaillée de tous ses champs :
