## Pour commencer {{ anchor: false }}

Pour mieux comprendre notre plateforme, il est crucial de reconnaître qu'elle se compose de deux structures distinctes : <a href="https://bo.jibril.immo" target="_blank">le backend</a>, responsable de la gestion du contenu, et <a target="_blank" href="http://jibril.immo">le front-end</a>, l'interface visible pour vos visiteurs et clients. # Technologies 

Nous avons utilisé deux technologies principales dans la construction du site web : <a href="https://react.dev/" target="_blank">React</a> et <a href="https://wordpress.org/" target="_blank">WordPress</a>. React, un framework JavaScript créé par <a href="https://www.facebook.com/" target='_blank'>Facebook</a>, est largement adopté par des grandes entreprises telles que <a href="https://airbnb.com" target='_blank'>Airbnb</a>, <a href="https://twitter.com" target='_blank'>Twitter</a> et <a href="https://netflix.com" target='_blank'>Netflix</a>. D'autre part, WordPress sert de système de gestion de contenu (CMS) robuste conçu pour une gestion efficace du contenu. Dans notre cas, nous exploitons la puissance de WordPress exclusivement en tant que plateforme backend en raison de son interface de tableau de bord familière sur le site web. Nous l'avons adapté pour répondre à nos besoins spécifiques en backend. React et WordPress communiquent entre eux via <a href="https://fr.wikipedia.org/wiki/Interface_de_programmation" target='_blank'>des API</a>. {{ className: 'lead' }}
