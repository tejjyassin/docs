# Composants de la page d'accueil

La page d'accueil du site web présente une collection soigneusement sélectionnée de composants, chacun conçu pour offrir une expérience engageante et conviviale à nos visiteurs. Plongeons dans les descriptions de ces composants : {{ className: 'lead' }}

## Image d'entête (Hero Image)

<Row>
  <Col>

    L'image d'entête est une grande image qui s'étend sur toute la largeur de la page. Elle est utilisée pour attirer l'attention sur la page et fournir une représentation visuelle du contenu de la page.

    ### Formulaire de recherche

    La section Hero inclut également un formulaire de recherche lié à elle. Cette recherche est configurée pour rechercher automatiquement une annonce en fonction de quelques termes de recherche déjà fournis. Ce formulaire est programmé de manière à ce que vous n'ayez pas besoin d'y apporter de modifications, mais si vous souhaitez changer le contenu du formulaire de recherche, vous pouvez le faire en accédant aux Paramètres globaux et en modifiant le contenu du formulaire de recherche.

  </Col>
  <Col>

  </Col>
</Row>

## Biens listés

<Row>
  <Col>

    Biens listés sont un groupe de biens répertoriés sur la page d'accueil. Cette section est configurée pour répertorier automatiquement les 6 biens les plus récents publiés sur le site web. Cette section est programmée de telle sorte que vous n'ayez pas besoin d'y apporter de modifications ! N'hésitez pas à consulter la page <a href="/biens">Biens</a> pour en savoir plus sur la manière d'ajouter des biens à votre site web.

    ### Paramètres disponibles

<Properties>
  <Property name="titre" type="text">
    Le titre de la section
  </Property>
  <Property name="type" type="sélection d'options">
    Le type de biens à sélectionner, les options disponibles sont Biens récents et Sélectionner des biens, ce qui vous permet de choisir les biens que vous souhaitez afficher sur la page d'accueil.
  </Property>
  <Property name="bouton" type="bouton">
    Le bouton à afficher sur la section, ce bouton est lié à la page des biens par défaut, mais vous pouvez changer le lien vers n'importe quelle page de votre choix.
  </Property>
  
</Properties>

  </Col>
  <Col>

    ![Paramètres des biens listés](/screenshots/biens-listee.png)*Paramètres des biens listés*
    ![Biens listés sur le site web](/screenshots/home-biens.png)*Biens listés sur le site web*

  </Col>
</Row>

## Bloc de description

<Row>
  <Col>

    Le bloc de description est une section simple qui vous permet d'ajouter un titre, une description et un bouton à votre page d'accueil. 

    ### Paramètres disponibles

<Properties>
  <Property name="titre" type="text">
    Le titre de la section
  </Property>
  <Property name="description" type="éditeur wysiwyg">
    Un bloc d'éditeur pour ajouter votre description et la manipuler comme bon vous semble
  </Property>
  <Property name="image" type="image">
    La section Image est facultative. Si elle est ajoutée, l'image sera affichée à gauche et le texte à droite. Sinon, le texte sera affiché au centre de la section. 
  </Property>
  <Property name="Bouton Plus" type="bouton">
    Un bouton libre à utiliser qui vous permettra de créer un lien vers n'importe quelle page de votre choix. Il vous suffit de définir le texte et le lien, et vous êtes prêt à partir.
  </Property>
  
</Properties>

  </Col>
  <Col>

    ![Paramètres du bloc de description](/screenshots/bloc-description.png)*Paramètres du bloc de description*
    ![Bloc de description sur le site web](/screenshots/home-bloc-description.png)*Bloc de description sur le site web*

  </Col>
</Row>

## Questions Fréquentes

<Row>
  <Col>

    Questions Fréquentes est une section permettant d'afficher une liste de questions et de réponses. Elle est utilisée dans ce cas pour conserver les mêmes mots-clés/contenu que nous avions précédemment sur l'ancien site web à des fins de référencement (SEO). Cette section est très flexible et peut être utilisée pour afficher n'importe quel type de contenu que vous souhaitez.

    ### Paramètres disponibles

<Properties>
  <Property name="titre" type="text">
    Le titre de la section
  </Property>
  <Property name="question" type="text">
    une section de texte pour afficher un titre/question 
  </Property>
  <Property name="réponse" type="éditeur wysiwyg">
  une section wysiwyg pour afficher la réponse à la question
  </Property>
  
</Properties>
