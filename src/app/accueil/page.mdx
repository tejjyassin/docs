# Home page components

The home page of the website features a carefully curated collection of components, each designed to provide an engaging and user-friendly experience for our visitors. Let's delve into the descriptions of these components: {{ className: 'lead' }}


## Hero Image

<Row>
  <Col>

    The hero image is a large image that spans the width of the page. It is used to draw attention to the page and provide a visual representation of the page's content.

    ### Search form

    The Hero section also include a search form linked to it, this search is set to automatically search for a listing based on a few search term already provided, this form is programmatically set that you don't need to make any adjustment to it, but if you want to change the content of the search form, you can do so by going to the Globale Settings and change the content of the search form.

## Bien listée

<Row>
  <Col>

    Bien listée is a group of listed biens on the home page, this section is set to automatically list the recent 6 biens that are published on the website, this section is programmatically set that you don't need to make any adjustment to it! Please feel free to review the <a href="/biens">Biens</a> page to learn more about how to add biens to your website.

    ### Available settings

<Properties>
  <Property name="title" type="text">
    The title of the section
  </Property>
  <Property name="type" type="select options">
    The type of biens to select, available options are Recent biens, and Select biens which allow you to select the biens you want to display on the home page.
  </Property>
  <Property name="bouton" type="boutton">
    The bouton to display on the section, this bouton is linked to the biens page by default, but you can change the link to any page you want.
  </Property>
  
</Properties>

  </Col>
  <Col>

    ![Bien listée Setting](/screenshots/biens-listee.png)*Biens listée settings*
    ![Biens listée on the website](/screenshots/home-biens.png)*Biens listée on the website*

  </Col>
</Row>

## Bloc de description

<Row>
  <Col>

    Bloc de description is a simple section that allow you to add a title, description and a button to your home page. 

    ### Available settings

<Properties>
  <Property name="titre" type="text">
    The title of the section
  </Property>
  <Property name="description" type="wyiwyg">
    An editor bloc to add your description and manipulate it however you feel
  </Property>
  <Property name="image" type="image">
    The Image section is optional, if added, the image will be displayed on the left side and the text on the right side, if not, the text will be displayed on the center of the section. 
  </Property>
  <Property name="Plus button" type="button">
    A free to use button that will allow you to link to any page you want. just set the text and the link and you are good to go.
  </Property>
  
</Properties>

  </Col>
  <Col>
